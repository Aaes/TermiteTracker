Compile java with javac HelloWorld.java

Compile header file with javah HelloWorld

Make C++ implementation HelloWorld.cpp

Compile C++ with:

g++ "-I/System/Library/Frameworks/JavaVM.framework/Versions/A/Headers" "-I/Developer/SDKs/MacOSX10.6.sdk/System/Library/Frameworks/JavaVM.framework/Versions/A/Headers" -c Main.cpp

Create shared library with
g++ -dynamiclib -o HelloWorldLib.jnilib HelloWorld.o

Run java program with java HelloWorld
